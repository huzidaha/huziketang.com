精心设计的学习计划

不需要再苦苦追寻学习一门技术最佳的方式
你只需要知道你想学什么
其他的担忧我们都帮你解决好了

HTML/CSS 13 课程

学习基本的设计、前端开发，开发用户体验良好的精美页面。

JavaScript 17 课程

最强大的脚本语言，用来构建强交互的 Web 页面。通往专业前端工程师必经之路。

React.js 2 课程

最热门的前端视图库，高效开发前端视图组件。

Vue.js 2 课程

数据驱动的 web 界面的渐进式框架。Vue.js 的目标是通过尽可能简单的 API 实现响应的数据绑定和组合的视图组件。

Angular.js 2 课程

AngularJS 是一款来自 Google 的前端 JavaScript 框架，简化了前端开发的负担，来帮助开发者从事web 开发。

PHP 2 课程

最流行的 Web 开发语言，很多网站采用 PHP 来进行搭建。掌握 PHP 让你可以建设多样的网站应用。

Python 4 课程

万能的胶水语言，几乎每个领域都会有 Python 的一席之地。人工智能、机器学习、Web 开发...

Rx.js 7 课程

RxJS 是一个强大的 Reactive 响应式编程库，提供了强大的数据流组合与控制能力。

Git 4 课程

Git 是一个开源的分布式版本控制系统，可以有效、高速的处理从很小到非常大的项目版本管理。掌握 Git 可以让你学会如何进行多人协作开发。

Database 3 课程

Take control of your application’s data layer by learning SQL, and take NoSQL for a spin if you’re feeling non-relational.

Electives 4 课程

Learn off the beaten Path and explore how these 课程 can enhance your overall development strategy.

胡子课堂如何打造专业开发者？

高效学习

提供高质量的视频教程和教学内容。实战经验丰富的讲师带领你一步一步由浅入深地进行学习，打造扎实的基础。

大量实战

全国首创的 Web 开发在线评测系统，你可以在浏览器中编写代码，获得即时的反馈。大量的测试用例保证你的代码符合实战标准，打造严密开发者思维。

收获成就

我们会根据你的学习的内容、获得的成就、徽章来对你进行综合评价和分析。并且据此给你制定更加合适的学习方案。
